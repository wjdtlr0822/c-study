int a =3;
a는 좌측값 3은 우측값
좌측값은 실제 주소가 있고, 우측값은 주소가없다.


참조자(레퍼런스)는 우측에 무조건 좌측값이 와야한다.
int &a = 3; //error
하지만 이를 가능하게 하기 위해서는
int &&a = 3; //success
&&를 써주면 된다.


template <typename T>
void test(T &a){} //를 할 경우 좌측값을 받지 못한다
받으려면 const T &a를 하면 좌측값을 받을 수 있지만 값을 변경할 수 없는 문제가있다.


int &a 는 좌측값 레퍼런스  // 좌측값만 받을 수 있다.
int &&a 는 우측값 레퍼런스 // 우측값만 받을수있다.
